// Was ist BYOK?

Dein eigener API Key – volle Kontrolle, null Overhead.

BYOK (Bring Your Own Key) bedeutet: Du verbindest Aura direkt mit einem KI-Anbieter deiner Wahl und nutzt dessen API. Du zahlst nur, was du wirklich verbrauchst – ohne monatliche Aura-Nachrichten zu verbrauchen.

BYOK ist in Aura eine optionale Alternative: Im Free-Plan erhältst du täglich 30 kostenlose Nachrichten. Wenn du darüber hinaus möchtest oder bestimmte Modelle brauchst, aktivierst du einfach deinen eigenen Key.

💡 Kurz erklärt: BYOK ist wie deine eigene SIM-Karte im fremden Handy – du nutzt die App, aber zahlst direkt beim Anbieter.
💸

Nur zahlen was du nutzt

Kein festes Abo beim Anbieter. Fange mit wenigen Euro an.

🤖

Alle großen Modelle

GPT-4o, Claude, Gemini, Llama, Mistral & mehr – du wählst.

🔒

AES-256 verschlüsselt

Dein Key wird sicher gespeichert und ist jederzeit wechselbar.

// Unterstützte Anbieter

Mit welchen Anbietern funktioniert BYOK?

Du kannst direkte API Keys der großen Anbieter nutzen – oder einen einzelnen OpenRouter-Key verwenden, um über 400 Modelle kostengünstig in einer App anzusprechen.

🚀

Direkte Anbieter

OpenAI (GPT-4o, o3), Anthropic (Claude), Google (Gemini), Mistral, xAI (Grok) und weitere – je ein Key pro Anbieter.

⭐

OpenRouter – der Alleskönner

Ein einziger Key für 400+ Modelle: GPT-4o, Claude, Llama, DeepSeek, Qwen & Co. – oft günstiger als direkt. Besonders empfohlen wenn du flexibel bleiben willst.

🔓

Open Source Modelle

Llama 3.3, DeepSeek-V3, Mistral Large, Qwen 2.5 und viele mehr – meist deutlich günstiger als proprietäre Modelle.

⭐ Unser Tipp: Starte mit OpenRouter – ein Account, ein Key, sofortiger Zugriff auf alle Modelle. Guthaben ab 5 €, keine monatlichen Kosten.
// Schritt-für-Schritt

API Key einrichten – in unter 5 Minuten.

01

Account erstellen

Gehe auf openrouter.ai → "Sign Up".

02

Guthaben aufladen

Klicke auf "Credits" und lade z.B. 5 € auf.

03

API Key erstellen

Gehe zu Settings → Keys → "Create Key".

04

Key einfügen

Key kopieren – wird nur einmal angezeigt. In Aura unter Einstellungen einfügen. 🎉

// Bereit?

Starte jetzt mit Aura & deinem eigenen Key.
