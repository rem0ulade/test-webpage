// Personas

🎭 Personas & @mentions

Eine Persona ist ein vordefinierter KI-Charakter mit eigenem System-Prompt, Tonalität und Expertise. Du rufst sie per @mention auf.

Jede Persona hat einen separaten Kontext. @coach erinnert sich an deine Fitness-Ziele, @developer kennt deinen Tech-Stack.

💡 Technisch: Personas werden als separate System-Prompts gespeichert und bei @mention automatisch in den API-Request eingebettet.

// Heartbeat Agent

⚡ Heartbeat Agent

Der Heartbeat Agent ist Auras autonomer Hintergrund-Prozess. Er läuft auf einem konfigurierbaren Zeitplan oder Event-Trigger – auch wenn du offline bist.

💡 Technisch: Cron-basiertes Scheduling auf Serverebene. Bei Auslösung wird ein vollständiger Aura-Kontext aufgebaut und als Push-Notification zugestellt.
// USE CASES – Was der Agent für dich erledigt
📈

Kurs-Monitoring

Überwacht täglich Aktien- oder Kryptopreise. Sobald ein Asset deinen Zielpreis erreicht, schickt dir Aura eine Push-Notification – du verpasst keine Chance mehr.

📬

API-Event → Telegram

Sobald ein Webhook-Event eintritt (z.B. neuer GitHub PR, Shopify-Bestellung, Fehler in deinem System), sendet Aura dir automatisch eine Nachricht per Telegram oder E-Mail.

🗓️

Täglicher Briefing-Agent

Jeden Morgen um 7 Uhr fasst der Agent deine Kalendertermine, offene Tasks und relevante News zusammen – direkt als Notification auf dein Handy.

🔁

Workflow-Automatisierung

Wiederholende Aufgaben wie Daten-Sync, Report-Generierung oder Content-Posting laufen automatisch auf deinem Schedule – ohne dass du dabei sein musst.

🌦️

Wetter & Reise-Alerts

Reist du morgen? Der Agent prüft Wetter, Zugverspätungen und gibt dir 12 Stunden vorher Bescheid – inklusive Empfehlung was du einpacken sollst.

🤖

Custom Skills

Definiere eigene Skills: "Wenn mein Server nicht antwortet, ruf mich an." Aura versteht natürliche Sprache und übersetzt sie in zuverlässige Hintergrund-Prozesse.
