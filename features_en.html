🎭 Personas & @mentions

Each persona has a separate context. @coach remembers your fitness goals, @developer knows your tech stack.

💡 Technical: Personas are stored as separate system prompts and automatically embedded into the API request on @mention.

// Heartbeat Agent

⚡ Heartbeat Agent

The Heartbeat Agent is Aura's autonomous background process. It runs on a configurable schedule or event trigger – even when you're offline.

💡 Technical: Cron-based scheduling on server level. On trigger, a full Aura context is built and delivered as a push notification.

Monitors stock or crypto prices daily. The moment an asset hits your target price, Aura sends you a push notification – never miss an opportunity again.
