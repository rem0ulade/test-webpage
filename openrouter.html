// Was ist OpenRouter?

Eine Fernbedienung für alle KI-Modelle.

OpenRouter vereint alle KI-Modelle unter einem Dach. Ein Account, einmal Guthaben aufladen – Zugriff auf über 400 KI-Modelle.

💡 Einfach erklärt: OpenRouter ist wie eine Universal-Fernbedienung für alle KI-Modelle.
💸

Nur zahlen was du nutzt

Kein Abo. Fange mit 5 € an.

🔓

Volle Kontrolle

Du entscheidest welches Modell.

🔒

Dein Key, deine Daten

Verschlüsselt gespeichert, jederzeit wechselbar.

// Schritt-für-Schritt

So holst du dir deinen API Key.

01

Account erstellen

Gehe auf openrouter.ai → "Sign Up".

02

Guthaben aufladen

Klicke auf "Credits" und lade z.B. 5 € auf.

03

API Key erstellen

Gehe zu Settings → Keys → "Create Key".

04

Key einfügen

Key kopieren – wird nur einmal angezeigt. In Aura unter Einstellungen einfügen. 🎉
